All layers are positive. Gerber file extension to layer mapping is as follows:

 *.GKO = board outline 
 *.GTS = top solder mask 
 *.GBS = bottom solder mask
 *.GTO = top silk screen 
 *.GBO = bottom silk screen 
 *.GTL = top copper
 *.G2L = inner layer 2 copper 
 *.G3L = inner layer 3 copper 
 *.G4L = inner layer 4 copper 
 *.G5L = inner layer 5 copper 
 *.G6L = inner layer 6 copper  
 *.G7L = inner layer 7 copper  
 *.GBL = bottom copper 
 *.XLN.0106 = layer 1-6 drill hits and sizes
 *.XLN.0716 = layer 7-8 drill hits and sizes
 *.XLN.0116 = layer 1-8 drill hits and sizes
